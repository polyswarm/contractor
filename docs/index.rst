What is Contractor
==================

Contractor is a tool to orchestrate deployment of PolySwarm's Solidity contracts over multiple networks.

It handles contract compilation, deployment, and automatic configuration of polyswarmd via consul.

It also contains useful debugging tools such as an interactive REPL and integration with various Solidity lints and analyses.
